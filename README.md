Project Overview
This project is a theme‑based interactive webpage built with HTML, CSS, and JavaScript.
It demonstrates a fixed navigation menu that changes style when scrolling and when hovering over menu items. The webpage follows a Travel & Adventure theme, showcasing destinations, adventures, and contact information with attractive visuals and gradients.

✨ Features
Fixed Navbar: Always visible at the top of the page.

Scroll Effect: Navbar background color changes when scrolling.

Hover Interactivity: Menu items highlight when hovered.

Theme‑based Sections: Home, Destinations, Adventures, and Contact.

Gradient Backgrounds: Each section has a unique gradient for visual appeal.

Images: Relevant placeholder images for each section.

Responsive Layout: Clean design that adapts well to different screen sizes.

🛠️ Technologies Used
HTML → Structure of the webpage

CSS → Styling, gradients, hover effects

JavaScript → Scroll detection and dynamic navbar behavior
